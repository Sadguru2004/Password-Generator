#Random Password Generator

A simple and responsive **Random Password Generator** built using **HTML, CSS, and JavaScript**.  
This project helps users generate strong and secure passwords with a single click.

---

## Features

- Generate strong random passwords
- Adjustable password length
- Includes:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- One-click password generation
- Copy password to clipboard
- Responsive and user-friendly UI

---

## Technologies Used

- **HTML** – Structure of the application  
- **CSS** – Styling and layout  
- **JavaScript** – Password generation logic
